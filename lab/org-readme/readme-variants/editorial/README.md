# Pitzi Labs

**Production that shows up when the need does.** `:>`

An infrastructure & operations practice, established 1997. Twenty-five years of
bare-metal data centers, 24×7 ops, and single-homed environments — now bridging
into cloud-native architecture by building it, breaking it, and operating it.

## What we do

- **Provision** — reproducible infrastructure as code, least-privilege by default.
- **Operate** — 24×7 on-call, multi-AZ resilience, observability that pages on signal, not noise.
- **Migrate** — bare-metal and single-homed environments into cloud-native, without the downtime theater.

## Stack

| Layer | Tooling |
|---|---|
| **Provisioning** | Terraform · Ansible |
| **Runtime** | AWS · Kubernetes · Docker · Linux |
| **Languages** | Bash · Python · Go |
| **Observability** | Prometheus · Grafana |
| **Data** | PostgreSQL |

## Operating principles

> least-privilege IAM · multi-AZ by default · WAFv2 attached · cost flagged before apply
